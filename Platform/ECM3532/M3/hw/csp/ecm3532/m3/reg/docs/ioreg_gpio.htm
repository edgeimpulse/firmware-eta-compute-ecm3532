0x0


BitsNameRWTitleDescription
31:0 COUNT RO M3 OSCILLATOR TICK COUNTS This register accumulates tick counts at the rate of the M3 oscillator. Firmware first issues a start command, which clears the count and then begins counting up. Firmware then issues a stop command which freezes the count so it can be read. Write 0x474F to clear and restart the counter. Write 0x4E4F to freeze the count. Write 0xAAAA to resume counting where it left off.
CLEAR_AND_START = 18255 First clear the counter and then begin counting M3 oscillator ticks.
FREEZE = 20047 Freeze the counter so it can be read.
RESUME = 43690 Resume counting from where it was stopped.

Description:
This register accumulates tick counts at the rate of the M3 oscillator. Firmware controls when it starts counting and when it stops counting. It is intended to measure the duration of firmware execution regions in core (M3) oscillator ticks.

31:24 RSVD1 RSVD Reserved Always write zero to reserved bit fields.
23:23 GPIO_PM_MODE RO STATE OF GPIO_PM MODE PIN This read only bit indicates the state of the GPIO PM MODE pin. If it is one then bootrom firmware must check GPIO[28]. If that is zero then bootrom firmware has to decde GPIO[31:29].
NORMAL = 0 Perform normal customer facing bootmode.
VALIDATION = 1 Select validdation bootmode from GPIO[31:29], provided GPIO[28] == zero.
22:22 FLUSH_FLASH RO DEBUGGER ATTACHED AND FLUSH FLASH REQUESTED This read only bit indicates that a debugger has been detected on the SWD interface while INFO page indicates that the FLASH is not to be preserved in this case. This bit is used in the boot ROM.
21:21 INFO_SECURE_BOOT RO FLASH INFO READER SECURE BOOT MODE This read only bit indicates that a secure boot protocol will be used at power up to provide a hardware root of trust state for the code to be executed in flash segment zero.
20:20 INFO_PRESERVE_FLASH RO FLASH INFO READER FLASH PRESERVED This read only bit indicates that the FLASH will be preserved when a debugger is attached to the SWD port. Zero indiates that it will be erased when a debugger is attached to hide the contents from view.
19:19 INFO_PRESERVE_SRAM RO FLASH INFO READER SRAM PRESERVED This read only bit indicates that the SRAM will be preserved when a debugger is attached to the SWD port. Zero indiates that it will be erased when a debugger is attached to hide the contents from view.
18:18 INFO_DEBUG_ENABLE RO FLASH INFO READER SWD DEBUG PORT ENABLED This read only bit indicates whether the SWD port is enabled for debug. Zero indiates that a debugger can not be connected through the SWD port.
17:17 INFO_CUST_UNLOCK RO FLASH INFO READER CUSTOMER UNLOCKED This read only bit indicates whether the customer section of the INFO page is unlocked for further programming. This is represents the state for the last 3072 bytes in the INFO page. If this bit is set to zero then INFO page can not be erased.
16:16 INFO_ETA_UNLOCK RO FLASH INFO READER ETA COMPUTE UNLOCKED This read only bit indicates whether the Eta Compute section of the INFO page is unlocked for further programming. This is represents the state for the first 1024 bytes in the INFO page. If this bit is set to zero then INFO page can not be erased.
15:15 DSP_FREQ_TOG RO STATE OF TOGGLE FOR DSP FREQ COUNTER CAPTURE A toggle on this read only bit indicates that the DSP frequency counter has produced a fresh value.
14:14 M3_FREQ_TOG RO STATE OF TOGGLE FOR M3 FREQ COUNTER CAPTURE A toggle on this read only bit indicates that the M3 frequency counter has produced a fresh value.
13:13 RTC_TMR_INT RO RTC TIMER INTERRUPT STATUS BIT This read only bit provides access to the RTC TIMER interrupt status bit.
12:12 RTC_FLAG RO RTC TOGGLE FLAG This read only bit toggles everytime the RTC time of day counter increments (once per second). It is used to guarantee safe reads of the current time of day.
11:11 UART1_RX_EMPTY RO UART1 RX FIFO EMPTY This read only bit tells driver level code whent there is at least one byte to the UART1 receiver FIFO.
10:10 UART1_TX_FULL RO UART1 TX FIFO FULL This read only bit tells driver level code whent there is room to add at least one more byte to the UART1 transmitter FIFO.
9:9 UART0_RX_EMPTY RO UART0 RX FIFO EMPTY This read only bit tells driver level code whent there is at least one byte to the UART0 receiver FIFO.
8:8 UART0_TX_FULL RO UART0 TX FIFO FULL This read only bit tells driver level code whent there is room to add at least one more byte to the UART0 transmitter FIFO.
7:0 WARM_START_MODE RO WARM START MODE FROM ALWAYS ON BLOCK These read only bits tell the BOOTROM how to warm start and if it should warm start. Anything greater than 0x8 is a warm start.
NORMAL_COLD_START = 0 Normal cold start in SRAM or flash.
FORCE_COLD_START_IN_FLASH = 1 Force cold start in flash even if valid SRAM program present.

Description:
Quick read of various external status bits.
