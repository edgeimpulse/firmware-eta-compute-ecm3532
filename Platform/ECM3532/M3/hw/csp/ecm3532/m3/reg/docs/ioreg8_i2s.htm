31:4 RSVD RSVD Reserved Always write zero to reserved bit fields.

RST = 1 Reset.
2:2 BIT64_MODE RW 64 BIT MODE Set this bit to set the frame size to 64 bits.
1:0 SAMPLE_FREQ RW SELECT SAMPLE FREQUENCY 00: 8ksps, 01: 16ksps, 10: 32ksps, 11: not implemented.
