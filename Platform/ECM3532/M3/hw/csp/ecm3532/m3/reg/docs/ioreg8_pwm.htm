PWM Generator


Put banner announcement here.


The PWM module contains four PWM generators. The PWM generators are configured with a command FIFO which sets their config, period, pulse start, and pulse end internal registers. A channel multiplexer steers PWM signals to any of the 16 output pins. The PWM supports synchronous channel changes to avoid glitches on the output.

31:30 SEL WO SELECTED PWM PWM generator selected for this command.
PWM0 = 0 Write to PWM0.
PWM1 = 1 Write to PWM1.
PWM2 = 2 Write to PWM2.
PWM3 = 3 Write to PWM3.
29:28 REG WO SELECTED PWM REGISTER PWM register selected for this command.
CFG = 0 Write to CFG register.
PER = 1 Write to PER register.
PSTRT = 2 Write to PSTRT register.
PEND = 3 Write to PEND register.
27:25 RSVD RSVD Reserved Always write zero to reserved bit fields.
24:24 WAIT WO WAIT FOR PROGRAM Write this command after the currently running program has finished.
EN = 1 Enable.
DIS = 0 Disable.
23:0 DATA WO FIFO DATA PWM register data. Derive format from the relevant CFG, PER, PSTRT, and PEND registers. There are helper enumerations below for setting bits in a CFG command.
CYCLE_INT = 524288 (For CFG command) Set bit-19 to trigger interrupt at the end of each PWM cycle.
PROG_INT = 262144 (For CFG command) Set bit-18 bit to trigger interrupt at the end of this program.
PULSE_DRIVE = 131072 (For CFG command) Set bit-17 for standard pulse output drive (Pulse is high), otherwise pulse is inverted (Pulse is low).
ONESHOT = 65536 (For CFG command) Set bit-16 to do only COUNT pulses and then stop. Disable this bit to run forever. CFIFO WAIT command will always load after COUNT pulses regardless of oneshot state.

Description:
Writes a command into the command FIFO. Writing the MSbyte commits all four bytes to the FIFO. Reading this register returns the next command waiting to come out of the FIFO. The WAIT field prevents the command at the top of the FIFO from being popped until the end of the currently running PWM program.
